# GoldenThread

GoldenThread es la evolución del proyecto desarrollado originalmente como TFG. Este nuevo repositorio tiene como objetivo reconstruir el proyecto desde cero, aplicando buenas prácticas, arquitectura moderna y una base tecnológica más sólida, para evitar que quede abandonado como ocurre con la mayoría de proyectos académicos.

El sistema se divide en dos partes principales:

1. **Frontend dinámico**  
   Una aplicación web moderna que permite navegar por el catálogo, filtrar productos, visualizar detalles y realizar acciones avanzadas que no estaban presentes en la versión original.

2. **API REST segura y escalable**  
   Un backend diseñado para ofrecer endpoints robustos, con un nivel de seguridad adecuado, evitando filtraciones de información sensible y garantizando una comunicación eficiente tanto con la base de datos como con la aplicación web.

GoldenThread busca convertirse en una plataforma completa, profesional y mantenible, que sirva como base para futuras mejoras y funcionalidades.


## Tecnologías utilizadas

### Frontend
   - Angular 17+
   - TypeScript
   - HTML5
   - CSS3 / SCSS
   - Angular Router
   - Angular HTTPClient
   - Reactive Forms
   - JWT Interceptor
   - RxJS
   - Angular CLI
   - PrimeNG / Angular Material (UI Components)

### Backend
   - Java 17
   - Spring Boot 3
   - Spring Web (REST API)
   - Spring Security + JWT
   - Spring Data JPA (Hibernate)
   - Maven
   - Lombok
   - Swagger / OpenAPI
   - ModelMapper / MapStruct

### Base de Datos
   - MySQL / PostgreSQL
